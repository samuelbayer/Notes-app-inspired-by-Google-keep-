# Note App

A personal note-taking app inspired by Google Keep.

## Features

- Create, edit, and delete notes
- Search notes with live filtering and debounce
- Filter notes between active and deleted (trash)
- View full note details in a popup
- Notes are saved in localStorage for persistence
- Automatic deletion of notes in the trash after 7 days
